Características Principales

Todo lo que necesitas para mantenerte organizado

Ejecución en Segundo Plano

Se ejecuta silenciosamente sin ventanas molestas. Solo aparece en la bandeja del sistema cuando lo necesitas.

Interfaz Web Moderna

Gestiona tus notificaciones desde una interfaz web elegante y fácil de usar. Sin instalaciones adicionales.

Notificaciones Programadas

Programa notificaciones para momentos específicos del día. Soporte para notificaciones recurrentes y únicas.

Almacenamiento Persistente

Tus notificaciones se guardan automáticamente. No perderás nada aunque reinicies tu computadora.

Notificaciones Nativas

Integración completa con el sistema de notificaciones de Windows. Aparecen como notificaciones del sistema.

Ligero y Rápido

Consume mínimos recursos del sistema. Desarrollado en Go para máximo rendimiento y eficiencia.

Cómo Funciona

Tres simples pasos para empezar

1

Descargar e Instalar

Descarga el ejecutable y ejecútalo. Se instalará automáticamente en la bandeja del sistema.

2

Configurar Notificaciones

Haz clic en el icono de la bandeja y selecciona "Agregar Notificación" para abrir la interfaz web.

3

¡Listo!

Tus notificaciones se ejecutarán automáticamente en los horarios programados.
